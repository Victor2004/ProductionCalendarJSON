# ProductionCalendarJSON

Консольное приложение для формирования JSON-файла с календарём на выбранный год с нерабочими, рабочими и сокращенными днями для пятидневной и шестидневной рабочих недель.

## Как использовать

Запуск командой `dotnet run` или файл по пути `./bin/Debug/net10.0/ProductionCalendarJSON`.
Прорамма попросит указать год и выбрать сервис или можно нажать ENTER для выбора варианта по умолчанию.
Можно запустить программу с параметрами, например:
```bash
dotnet run -y 2026 -s isdayoff.ru
```

### Параметры командной строки
| Параметр    | Краткая форма | Описание                                                                            |
| :---------- | :------------ | :---------------------------------------------------------------------------------- |
| `--year`    | `-y`          | Год для генерации календаря (от 2000 до 2100). Если не указан, будет запрошен       |
| `--service` | `-s`          | Имя сервиса для получения данных (по умолчанию: `isdayoff.ru`)                      |
| `--output`  | `-o`          | Путь для сохранения JSON-файла                                                      |
| `--help`    | `-h`          | **ВАЖНО:** Показ справки работает только при прямом запуске скомпилированного файла |
